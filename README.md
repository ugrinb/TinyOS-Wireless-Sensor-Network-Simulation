# Wireless Sensor Network Simulation with TinyOS & TOSSIM

This repository contains a reconstructed version of my 2016 academic project focused on **Wireless Sensor Networks (WSN)**.

## 📝 Overview
The project explores the architecture of **TinyOS**, an event-driven operating system designed for low-power wireless devices. It demonstrates the use of the **nesC** programming language and the **TOSSIM** simulator to model sensor node behavior.

## 🚀 Key Features
- **Simulation Control:** Uses a **Python script** (`testblink.py`) to initialize the simulation environment, manage node boot sequences, and capture debug events.
- **Event-Driven Programming:** Implementation of timer-based interrupts in nesC.
- **TOSSIM Integration:** Capturing real-time execution data of a virtual sensor node (Mote).

## 🛠️ Technical Depth (for Cybersecurity/IoT)
This project highlights my foundational understanding of:
- **Resource-Constrained Systems:** Working with limited RAM/CPU environments (TinyOS uses only a few KBs).
- **Network Layer Basics:** Understanding how wireless nodes interact at a low level, which is critical for **IoT Security** and **Threat Analysis**.
- **Automation:** Using Python to bridge high-level control with low-level embedded systems.
